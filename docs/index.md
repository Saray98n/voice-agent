# 🎙️ Software Journey - Voice Agent

## Introducción

Voice Agent es un proyecto académico desarrollado para comprender la arquitectura interna de los agentes conversacionales por voz utilizando Python y programación asíncrona mediante asyncio.

El sistema implementa un flujo conversacional compuesto por:

Micrófono → Speech-to-Text → LLM → Text-to-Speech → Altavoz

Durante el desarrollo se aplicaron metodologías modernas de ingeniería de software apoyadas por inteligencia artificial utilizando Claude Code, GitHub Issues, Ralph Loops y revisiones arquitectónicas iterativas.

## Estado Final del Proyecto

El proyecto finaliza con una arquitectura modular documentada, un frontend demostrativo funcional, una implementación backend basada en Python y asyncio, una estrategia de gestión mediante GitHub Issues, documentación de handoffs para gestión de contexto y una revisión arquitectónica intermedia basada en Ralph Loops.

La solución desarrollada permite comprender el ciclo completo de diseño, implementación, revisión arquitectónica y documentación de un sistema asistido por inteligencia artificial.
